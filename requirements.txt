# ============================================================
# Notes
# ============================================================
# 1. tantivy is required ONLY when using SearchType.keyword in LanceDB.
#    If you switch to SearchType.vector or SearchType.hybrid, tantivy
#    is not strictly required (but doesn't hurt to keep it installed).
#
# 2. pylance and lancedb MUST be installed together — pylance is the
#    Python interface that phidata uses to communicate with LanceDB.
#
# 3. The SQLite storage backend (SqlAgentStorage) uses Python's built-in
#    sqlite3 module — no extra install required.
#
# 4. On Google Colab, /tmp/lancedb is DELETED when the session ends.
#    Mount Google Drive and point LanceDB uri to Drive for persistence.
#    Example: uri="/content/drive/MyDrive/lancedb"
#
# 5. duckduckgo-search v5+ changed its API. If you see errors, ensure
#    you have version 5.0.0 or higher:
#    pip install duckduckgo-search --upgrade
